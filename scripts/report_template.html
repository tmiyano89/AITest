📊 メトリクス定義

正解率 (Accuracy Rate)

抽出すべき項目のうち、正しく抽出された項目の割合。値の正確性も含む。

誤り率 (Error Rate)

抽出すべき項目のうち、間違った値で抽出された項目の割合。

欠落率 (Missing Rate)

抽出すべき項目のうち、全く抽出されなかった項目の割合。

過剰抽出率 (Unexpected Rate)

抽出すべきでない項目が抽出された割合。

Precision (精度)

抽出された項目のうち、正しい項目の割合。TP / (TP + FP)

Recall (再現率)

抽出すべき項目のうち、実際に抽出された項目の割合。TP / (TP + FN)
