📋 メトリクス定義

精度メトリクス

  • 正解率 (Correct Rate): 抽出すべき項目のうち、正しく抽出された項目の割合
  • 誤り率 (Wrong Rate): 抽出すべき項目のうち、値が間違って抽出された項目の割合
  • 欠落率 (Missing Rate): 抽出すべき項目のうち、抽出されなかった項目の割合
  • 過剰抽出率 (Unexpected Rate): 抽出すべきでない項目が抽出された割合（100%を超えることがある）
  • Precision: 抽出した項目のうち、正しい項目の割合
  • Recall: 抽出すべき項目のうち、実際に抽出された項目の割合

抽出時間メトリクス

  • 平均抽出時間: 全テストケースの抽出時間の平均値
  • 最小抽出時間: 全テストケースの抽出時間の最小値
  • 最大抽出時間: 全テストケースの抽出時間の最大値
  • 総抽出時間: 全テストケースの抽出時間の合計
